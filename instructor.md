# Mensaje del instructor

Este archivo fue agregado directamente desde GitHub mientras ustedes trabajaban localmente.

## ¿Qué acaba de pasar?

Tu repositorio en GitHub tenía un archivo nuevo que tu PC no tenía.
Cuando corriste `git fetch` y `git pull` — Git trajo ese archivo a tu carpeta.

Eso es exactamente lo que pasa cuando trabajás en equipo:

- Un compañero sube cambios a GitHub
- Tu PC queda desactualizado
- `git pull` sincroniza todo

## Regla para el trabajo en equipo

```
Antes de empezar a trabajar → git pull
Después de terminar → git push
```

Así el historial siempre está limpio y evitás conflictos.
